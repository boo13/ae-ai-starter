# Verified AE Data

Empirically verified After Effects match-names, property types, enum values,
and runtime behavior. All data was generated by running discovery scripts
inside After Effects — nothing is guessed.

## For AI Assistants

Before writing any script that applies an AE effect or accesses shape layer
properties, check this directory first. See the "Verified AE Data" section
in CLAUDE.md for the full workflow.

## Directory Structure

- `effects/` — One JSON file per verified effect (e.g., `ADBE-AudSpect.json`)
- `properties/` — Property trees for non-effect elements (e.g., `shape-layer.json`)
- `gotchas.md` — Runtime pitfalls and lessons learned
- `tools/` — Discovery and diagnostic scripts (maintainer use)

## For Maintainers

To add a new effect to the database:

1. Open `tools/discover_effect.jsx` in AE (File > Scripts > Run Script File)
2. Enter the effect match-name when prompted
3. The script writes a JSON file to `effects/`
4. Review the generated JSON and fill in `knownAliases` and `warnings`
5. Commit the JSON file

To verify shape layer properties:

1. Run `tools/discover_shape_properties.jsx` in AE
2. Output goes to `properties/shape-layer.json`
3. Commit the JSON file
