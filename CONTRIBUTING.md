# Contributing

Thanks for your interest in contributing!

## Reporting Issues

Open a [GitHub issue](../../issues) and describe:
- What you expected to happen
- What actually happened
- Your After Effects version and OS

## Suggesting Changes

Open an issue first to discuss the idea before writing code. This saves everyone time.

## Submitting a Pull Request

1. Fork the repo and create a branch from `main`
2. Make your changes
3. Open a pull request with a clear description of what changed and why

## Code Style

All scripts must be ES3-compatible ExtendScript — no `let`, `const`, arrow functions, or template literals. See the constraints in `AGENTS.md`.
